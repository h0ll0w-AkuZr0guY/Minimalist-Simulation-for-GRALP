# 项目介绍：Minimalist-Simulation-for-GRALP

## 1. 项目概述

Minimalist-Simulation-for-GRALP 是一个基于 PyBullet 物理引擎的机器人仿真平台，结合了近端策略优化（PPO）算法，用于训练和测试自主导航机器人。该项目采用了模块化设计，便于扩展和学习，适合零基础小白入门强化学习和机器人仿真。

## 2. 项目目标

- 提供一个简洁、高效的机器人仿真环境
- 实现基于 PPO 算法的自主导航控制
- 展示现代化的深度学习架构在机器人领域的应用
- 为初学者提供系统化的学习路径

## 3. 项目结构

```
Minimalist-Simulation-for-GRALP/
├── robot.py          # 机器人模型和控制
├── scene.py          # 仿真场景管理
├── camera_ctrl.py    # 相机控制
├── config_loader.py  # 配置加载
├── task.py           # 主仿真循环
├── ppo_api/          # PPO 算法实现
│   ├── encoder.py    # 光线编码器
│   ├── inference.py  # 推理逻辑
│   └── ppo_models.py # PPO 模型定义
└── learning/         # 学习文档（新建）
```

## 4. 核心组件

### 4.1 机器人系统 (`robot.py`)
- 创建和控制差速驱动机器人
- 模拟激光雷达传感器
- 提供机器人状态获取接口

### 4.2 仿真场景 (`scene.py`)
- 动态和静态障碍物管理
- 环境边界设置
- 目标点生成

### 4.3 相机控制 (`camera_ctrl.py`)
- 鼠标交互控制
- 视角调整
- 场景可视化

### 4.4 配置管理 (`config_loader.py`)
- PPO 算法参数配置
- 仿真环境参数配置
- 动态加载和合并配置

### 4.5 主仿真循环 (`task.py`)
- 机器人导航任务
- 碰撞检测
- 目标点采样和更新

### 4.6 PPO 算法实现 (`ppo_api/`)
- 光线编码器：处理激光雷达数据
- 策略网络：生成机器人动作
- 价值网络：评估状态价值
- 推理逻辑：模型部署和推理

## 5. 技术栈

- **物理引擎**：PyBullet
- **深度学习框架**：PyTorch
- **算法**：PPO (Proximal Policy Optimization)
- **模型格式**：ONNX
- **推理引擎**：ONNX Runtime

## 6. 关键技术亮点

### 6.1 光线编码技术
- 深度可分离卷积
- 注意力机制
- Squeeze-and-Excite 模块

### 6.2 高效的 PPO 实现
- 共享编码器架构
- tanh 压缩的动作空间
- 稳定的策略更新

### 6.3 模块化设计
- 清晰的代码结构
- 低耦合的组件
- 易于扩展和修改

## 8. 开始使用

### 8.1 安装依赖

```bash
# 创建虚拟环境
python -m venv .venv

# 激活虚拟环境
# Windows
.venv\Scripts\activate
# Linux/Mac
source .venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```

### 8.2 运行仿真

```bash
python task.py
```
